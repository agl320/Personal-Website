DEVPOST_SCRAPER

STATUS : COMPLETE

Python scraper that collects submission data from any given number of Devpost hackathon pages and returns project data.

Originally forked from Lynn Samson's Python Devpost Scraper, I implemented an improved description retrieval system using extractive summarization through spaCy, added multiple URL support, and the ability to specify the number of submissions from each page of each hackathon.

  • spaCy
  • NLP
  • Flask
